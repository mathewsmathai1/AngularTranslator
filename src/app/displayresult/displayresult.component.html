Translator Results

********This Translator is powered by IBM Watson********

The API calls used is an abstraction over the actual IBM API calls. It uses a predefined instance of the IBM service.
Following are the API Calls:
  1. List Of Languages: https://therealtranslator.herokuapp.com/languages/ |Response: Array Of Json Objects (language, language_name)
  2. Translate: https://therealtranslator.herokuapp.com/translate?source=[language]&target=[language]&text=AddSourceAndTargetLanguage |Response: Text
********Use the above calls to intergrate the translator with your application********
